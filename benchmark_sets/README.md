# Benchmark Sets

Reference benchmark sets used for evaluation and figure reproduction.

## LCL

- [`lcl/lcl_1852_peak_ids.tsv`](./lcl/lcl_1852_peak_ids.tsv): finalized LCL ATAC-seq benchmark peak set with `region_id`, `chr`, `start`, and `end`.
- [`lcl/hg_tissues.json`](./lcl/hg_tissues.json): the 44 AlphaGenome donor-specific LCL ontology terms (`EFO:0010841`–`EFO:0010889`) with their HG donor ID mapping, used as `ontology_terms` when querying the AlphaGenome API for personal LCL ATAC tracks.
